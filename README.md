# File Upload & Download Web App

A web application built using Python Flask and MongoDB Atlas that allows users to upload, download, and delete files.

## Made By
- Name: Your Name
- Class: Second Year
- Subject: Python Programming

## Features
- Upload any file from your computer
- Download uploaded files
- Delete files
- File details like name, size and date saved in MongoDB

## Technologies Used
- Python
- Flask
- MongoDB Atlas
- HTML & CSS

## How to Run

**Step 1 - Install libraries:**
```
pip install flask pymongo dnspython
```

**Step 2 - Run the app:**
```
python app.py
```

**Step 3 - Open browser and go to:**
```
http://127.0.0.1:5000
```
